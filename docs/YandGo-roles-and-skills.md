## Responsibilities

## Software Engineer

Design, develop, test, and maintain software applications

Write clean, efficient, and well-documented code

Collaborate with product managers, designers, and other engineers

Debug, troubleshoot, and fix bugs and performance issues

Participate in code reviews and follow best engineering practices

Contribute to system improvements and feature enhancements

## Security / Risk Engineer

Identify, assess, and mitigate security risks across systems and applications

Implement security controls, monitoring, and incident response processes

Conduct vulnerability assessments, penetration testing, and threat modeling

Ensure compliance with security standards and regulations (e.g., ISO, SOC 2, GDPR)

Investigate security incidents and recommend remediation actions

Work with engineering teams to embed security best practices into development

## Back-End Engineer

Build and maintain server-side applications, APIs, and services

Design and optimize databases and data storage solutions

Ensure application performance, scalability, and reliability

Integrate third-party services and internal systems

Implement authentication, authorization, and data protection mechanisms

Collaborate with front-end developers and DevOps teams

## System Architect

Design high-level system architecture and technical solutions

Define technology standards, frameworks, and system integration patterns

Ensure scalability, reliability, security, and performance of systems

Translate business requirements into technical architectures

Review and guide development teams on architectural decisions

Evaluate and recommend new technologies and platforms

## Data Engineer

Design, build, and maintain data pipelines and ETL processes

Manage large-scale data storage systems (data warehouses, data lakes)

Ensure data quality, integrity, and availability

Optimize data processing for performance and scalability

Collaborate with data scientists and analysts to support analytics needs

Implement data governance, security, and compliance practices
